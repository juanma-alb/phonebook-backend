**Phonebook Backend**

A RESTful API built with Node.js and Express for managing phonebook entries. This project is part of the **Full Stack Open** course (Part 3).

 You can see my main repository [here](https://github.com/juanma-alb/fullstackopen).


##  Technologies Used

- **Node.js** - JavaScript runtime environment.
- **Express** - Web framework for Node.js.
- **Nodemon** - Development tool to automatically restart the server upon any change.
